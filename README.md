# Quant-Investor

一套完整的量化投资研究框架，从原始数据到交易决策，涵盖数据获取、因子挖掘、模型训练、风险控制、宏观判断、AI决策六个层面。

## 它能做什么

这套系统适合需要系统性研究股票市场的场景：

- **量化研究员**：快速验证因子有效性，自动完成IC测试、分层回测
- **基金经理**：实时监控组合风险，获得仓位调整建议
- **个人投资者**：获取基于AI的投资决策参考，了解市场宏观环境

## 系统架构

整个系统按照数据流动的自然顺序分为六层：

### 第一层：数据层

处理所有原始数据的获取和清洗。

数据来源包括Tushare（A股）、yfinance（美股）、FRED（宏观数据），覆盖中、港、美、欧、日五个主要市场。数据清洗环节会做去极值、缺失值填充、标准化处理，确保进入下游的因子计算时质量可靠。

### 第二层：因子层

基于清洗后的数据计算各类因子，并验证其有效性。

系统内置了Alpha158因子库，包含130多个常用因子，涵盖动量、波动率、成交量、均值回归等维度。每个因子都会经过IC分析（看因子与未来收益的相关性）、分层回测（看因子分组是否单调）、换手率分析（看因子的稳定性），最后给出综合评分帮助筛选。

### 第三层：模型层

用机器学习模型预测股票未来收益。

支持Random Forest、XGBoost、SVM、LSTM四种模型。训练时使用时间序列交叉验证避免数据泄露，支持Optuna自动调参找到最优超参数，训练完成后可以用SHAP分析模型的特征重要性，理解决策逻辑。模型版本通过MLflow管理，方便回溯和对比。

### 第四层：宏观层

判断当前市场环境，作为风控的第一道防线。

针对A股监控两融余额、GDP增速、巴菲特指标（市值/GDP）、M1-M2剪刀差等；针对美股监控联邦利率、失业率、Shiller PE、期限利差、VIX等。综合这些指标给出红/黄/绿/蓝四档信号，红灯表示高风险应减仓，绿灯表示可以积极布局。

### 第五层：风控层

在投资组合层面做风险控制。

计算VaR（风险价值）、CVaR（条件风险价值）、夏普比率、最大回撤等指标。支持多种压力测试场景：2008年金融危机、2015年A股熔断、2020年疫情暴跌。仓位管理会根据宏观信号动态调整，红灯时仓位降至30%，绿灯时可以提到80%。

### 第六层：决策层

AI多模型辩论，输出最终投资建议。

五个专业模型分别从不同角度分析：财务模型看ROE和估值，行业模型看竞争格局，宏观模型看经济周期，技术模型看价格趋势，风险模型评估潜在损失。每个模型都给出看多/看空观点和置信度，最后综合形成投资决策：买不买、买多少、目标价、止损位。

## 主要特性

**工程化**：59个单元测试覆盖核心逻辑，使用pytest + GitHub Actions做持续集成，代码遵循PEP 8规范。

**回测系统**：基于Backtrader实现事件驱动回测，包含滑点、手续费、印花税、市场冲击成本等完整交易成本模型，支持因子中性化处理（市值/行业）。

**可扩展**：采用依赖注入降低模块耦合，Redis缓存热点数据，异步处理提高LLM调用吞吐量。架构上支持微服务拆分，可以水平扩展到多节点。

**合规性**：内置风险管理政策框架，支持生成Form PF（美国）、AIFMD（欧盟）、CSRC（中国）三种格式的合规报告，完整的操作审计日志。

## 快速开始

安装依赖：

```bash
git clone https://github.com/alpha-mw/myQuant.git
cd myQuant
pip install -r requirements.txt
```

配置API密钥：

```bash
cp .env.example .env
# 编辑.env，填入Tushare token和OpenAI API key
```

运行完整分析：

```python
from scripts.unified.v72_full_market_analysis import run_full_analysis

result = run_full_analysis()
print(result['top_15_stocks'])  # 查看推荐的15只股票
```

启动风险监控面板：

```bash
streamlit run scripts/unified/risk_dashboard.py
```

## 注意事项

这套系统生成的分析报告仅供参考，不构成投资建议。模型预测基于历史数据，无法保证未来收益。使用者需要理解量化投资的风险，根据自身情况做出独立判断。

系统需要一定的计算资源，全市场1900只股票的分析在普通服务器上大约需要30-60分钟。建议先从小股票池开始测试。
