## 资金杠杆与情绪 (Leverage)

### 指标定义

- **两融余额**: 融资融券余额总量（万亿元），反映市场杠杆资金规模
- **两融/流通市值比**: 两融余额占A股流通市值的比例（%），反映杠杆资金相对规模

### 状态判断规则

| 两融/流通市值比 | 状态 | 信号 |
|:---|:---|:---|
| > 4.0% | 极度疯狂 | 🔴 强烈风险预警 |
| 3.0% - 4.0% | 偏热 | 🟡 注意风险 |
| 2.0% - 3.0% | 结构健康 | 🟢 正常 |
| 1.5% - 2.0% | 偏冷 | 🟡 关注机会 |
| < 1.5% | 极度冷清 | 🔵 底部区域 |

### 历史大顶对标

- **2015年疯牛顶**: 两融余额 2.27万亿，占流通市值比 4.5%
- 当两融余额接近或超过2.27万亿，且占比接近4.5%时，发出强烈风险预警

### 分析要求

1. 获取最新两融余额数据
2. 获取A股流通市值数据
3. 计算两融/流通市值比
4. 与2015年牛市顶部数据对标
5. 输出状态判断和风险等级

## 经济景气度 (Growth)

### 指标定义

- **GDP同比增速**: 中国GDP同比增长率（%），反映经济整体景气度

### 状态判断规则

| GDP同比增速 | 状态 | 说明 |
|:---|:---|:---|
| > 6.0% | 高速增长 | 经济过热风险 |
| 5.0% - 6.0% | 稳健增长 | 经济健康 |
| 4.0% - 5.0% | 中速增长 | 增速放缓 |
| 3.0% - 4.0% | 低速增长 | 经济下行压力 |
| < 3.0% | 增长乏力 | 衰退风险 |

### 分析要求

1. 获取最新季度GDP同比增速
2. 标注数据所属季度（如2025Q3）
3. 与前几个季度趋势对比
4. 判断经济周期位置

## 整体估值锚 (Valuation)

### 指标定义

- **A股总市值**: A股所有上市公司总市值（万亿元）
- **年度GDP预估**: 当年GDP预估值（万亿元）
- **市值/GDP（巴菲特指标）**: A股总市值与年度GDP的比值（%），衡量市场整体估值水平

### 状态判断规则

| 巴菲特指标 | 状态 | 信号 |
|:---|:---|:---|
| > 120% | 极度高估 | 🔴 泡沫区域 |
| 100% - 120% | 估值偏高 | 🟡 注意风险 |
| 80% - 100% | 合理偏高 | 🟡 谨慎乐观 |
| 60% - 80% | 合理区间 | 🟢 正常 |
| 40% - 60% | 低估区间 | 🟢 安全区域 |
| < 40% | 极度低估 | 🔵 历史底部 |

### 历史大顶比值参考

- **2007年疯牛顶**: 巴菲特指标 ~125%
- **2015年疯牛顶**: 巴菲特指标 ~110%
- **底部安全区间**: 40% - 60%

### 分析要求

1. 获取A股总市值数据
2. 获取或估算当年GDP
3. 计算巴菲特指标
4. 与2007年、2015年历史大顶对标
5. 判断当前估值位置
6. 标注是否处于安全区间

## 通胀与货币 (Inflation & Money)

### 指标定义与状态判断

#### CPI同比
- 反映消费品价格变动，衡量通胀水平
- **> 3%**: 通胀偏高 🟡
- **1% - 3%**: 温和通胀 🟢
- **0% - 1%**: 低通胀 🟡
- **< 0%**: 通缩 🔴

#### PPI同比
- 反映工业品出厂价格变动，衡量工业价格趋势
- **> 5%**: 工业品价格过热 🔴
- **0% - 5%**: 工业价格正常 🟢
- **-3% - 0%**: 工业价格下行 🟡
- **< -3%**: 工业通缩 🔴

#### M1-M2剪刀差
- M1增速减去M2增速，反映资金活化程度
- **> 0**: 资金活化，经济活跃 🟢
- **-3% - 0%**: 轻度存款定期化 🟡
- **< -3%**: 存款定期化严重 🔴（资金避险，企业投资意愿低）

#### M2增速
- 广义货币供应量增速，反映央行印钞速度
- **> 10%**: 宽松 🟢（利好股市）
- **8% - 10%**: 适度 🟡
- **< 8%**: 偏紧 🔴

#### 社融当月增量
- 社会融资规模当月新增量（亿元），反映信用扩张/收缩
- 需与历史同期对比判断：
  - **显著高于历史同期**: 信用扩张 🟢
  - **接近历史同期**: 信用平稳 🟡
  - **显著低于历史同期**: 信用收缩 🔴

### 分析要求

1. 获取最新月度CPI同比数据
2. 获取最新月度PPI同比数据
3. 获取最新M1、M2增速数据，计算M1-M2剪刀差
4. 获取最新社融当月增量数据
5. 每个指标独立判断状态
6. 综合评估货币环境对股市的影响

## 综合风控信号

根据四大模块的状态，生成综合风控信号：

| 综合信号 | 触发条件 | 建议 |
|:---|:---|:---|
| 🔴 高风险 | 任意2个模块出现红色信号 | 降低仓位，防御为主 |
| 🟡 中风险 | 任意2个模块出现黄色信号 | 控制仓位，精选个股 |
| 🟢 低风险 | 多数模块为绿色信号 | 正常配置，积极布局 |
| 🔵 极低风险 | 多数模块处于底部/低估区域 | 加大配置，逆向布局 |

## 数据获取方法

使用 `scripts/v2.5/cn_macro_data/macro_risk_terminal.py` 脚本获取所有指标数据。

```python
from scripts.v2.5.cn_macro_data.macro_risk_terminal import MacroRiskTerminal

terminal = MacroRiskTerminal(tushare_token="YOUR_TOKEN")
report = terminal.generate_risk_report()
```

数据源优先级：Tushare > AKShare > 手动输入

每个指标的数据获取、计算过程和判断逻辑必须在报告中完整展示，确保用户能理解分析过程。
